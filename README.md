# Mini Sistema de Cadastro em Python

Projeto desenvolvido para praticar lógica de programação utilizando Python.

## Funcionalidades
- Menu interativo
- Mostrar nome do usuário
- Mostrar idade
- Soma de dois números
- Opção de sair do sistema

## Tecnologias
- Python 3

## Como executar o projeto
1. Baixe o arquivo `main.py`
2. Execute no terminal com:
   python main.py

## Objetivo
Projeto criado para estudo e portfólio, focado em:
- Estruturas condicionais
- Laços de repetição
- Entrada e saída de dados
